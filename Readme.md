### 2019-12-22
- Use key h,j,k,l
  - h : Move left
  - j : Move down
  - k : Move up
  - l : Move right
- Use key w,e,b
  - w : Move next word first character
  - e : Move current word last character or next word last character
  - b : Move before word first character
